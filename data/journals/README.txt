# Getting list of biology-related papers

1. Select all papers that contain the word "bio*" OR "med*" NOT "biog*" in Unpaywall
2. Get list of bio-related journals from wikipedia (https://en.wikipedia.org/wiki/List_of_biology_journals) [https://gist.github.com/universvm/d30c7e59addca585db8c6b1828dd4d1b]
3. Add all papers from the wikipedia journals contained in unpaywall 
5. Remove duplicates

# Getting list of journals categories

1. Go through wikipedia categories https://en.wikipedia.org/wiki/List_of_biology_journals
2. Remove special characters and "," from journal names
3. Append journals to each category
